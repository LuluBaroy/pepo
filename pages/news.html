Aide à la navigation

RACCOURCIS CLAVIER

  • CTRL + F : Je cherche un mot particulier sur la page, je clique simultanément sur CTRL + F et j’écris le mot en question dans l’encadré qui s’ouvre (en bas à gauche du navigateur).
  • TAB : Je peux utiliser la touche pour naviguer sur le site en passant de lien en lien.
  • CTRL + C : Je copie un texte qui m’intéresse.
  • CTRL + V : Je colle le texte que j’ai copié à un endroit voulu.
  • CTRL + clic gauche de la souris : J’ouvre un lien dans un nouvel onglet.
  • CLIC MAINTENU sur la roulette de la souris + MOUVEMENT de la main vers le haut ou le bas : Je fais défiler la page internet vers le haut/le bas, sans faire tourner la roulette de la souris.
  • ENTRÉE : Je valide la sélection d’un lien et je l’ouvre.

STYLES ALTERNATIFS

  • CTRL + + ou CTRL + faire rouler la roulette en avant : Je zoome sur la page, j’agrandis donc l’écriture.
  • CTRL + - ou CTRL + faire rouler la roulette en arrière : Je dézoome sur la page, je rétrécis donc l’écriture.
  • MODE NUIT (lune) : Je change le style du site pour obtenir des couleurs plus contrastées et avoir plus de facilité à lire le contenu.

LECTEUR D’ÉCRAN Les fiches sont lisibles avec un logiciel de lecture. Le texte des fiches est écrit dans le code html du site et est donc lisible par le lecteur d’écran.

COMMANDE VOCALE
Le nom du site se prononce phonétiquement : [w] [i] [k] [a] [i] [a] (ou en langage « ça se prononce comme ça s’écrit » : ouikaïa)
