Aide à la navigation

RACCOURCIS CLAVIER

  • CTRL + F : Recherche par mot(s)
  • TAB : Navigation clavier
  • CTRL + C : Copier
  • CTRL + V : Coller
  • CTRL + clic gauche : Ouvrir dans un nouvel onglet
  • CLIC roulette + MOUVEMENT vers le haut/bas : Défilement
  • ENTRÉE : Ouverture d'un lien

STYLES ALTERNATIFS

  • CTRL + + ou CTRL + roulette avant : Zoom +
  • CTRL + - ou CTRL + roulette arrière : Zoom -
  • Clic sur la lune : Mode Nuit
